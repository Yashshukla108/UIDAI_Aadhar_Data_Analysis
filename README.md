Aadhaar Data Analysis Project

This repository contains a data analysis project based on UIDAI Aadhaar enrolment demographic updates and biometric updates data.

Project Objectives
- Analyze Aadhaar enrolment trends
- Identify age wise and regional patterns
- Use update data as proxy indicators for migration
- Provide policy relevant insights for governance

Contents
- Aadhaar_Analysis.ipynb Google Colab notebook
- Final_Report.pdf 15 page analytical report
- visuals folder containing charts and maps

Tools Used
Python
Pandas
Matplotlib
GeoPandas
Google Colab

Note
All analysis is performed on aggregated and anonymised data
